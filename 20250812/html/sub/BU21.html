솔루션 부문

R&D 과제

수소 기반 미래 에너지 기술을 선도하는 넥서스가스의 R&D 비전

고도화된 수소 에너지 인프라 실현을 위해 다양한 연구개발 과제를 수행하고 있습니다.

수소 저장·운송·충전 전 주기를 아우르는 인프라 기술부터, 안전 제어 시스템, 디지털 트윈 기반의
설비 운영 기술까지 실제 사업 적용이 가능한 실증형 기술을 중심으로 R&D를 추진합니다.

핵심 기능

실시간 현장 3D 기사화

설비, 배관, 건물 등 전체 인프라를
3D로 생생하게 시각화

센서·설비 상태 모니터링

온도·압력·유량 등 실시간
센서 데이터를 3D에 연동 표시

이상징후 감지 및 알람

시중치를 벗어나면 즉시 경고,
시각/청각 알람으로 사고 예방

공정 시뮬레이션

저장부터 충전까지 공정 흐름을
가상으로 재현 및 훈련 지원

디지털 트윈 시나리오

수소 수급/운송 시작

01

수소 수급/운송 시작
  • 외부 차량 입고 인식, 도착 상태 자동 표기​
  • 실시간 위치·온도·압력 정보
    3D 모델에 연동 표시​
  • 예시: 차량 아이콘 점멸 + 위치좌표 박스 노출

저장탱크 상태 확인

02

저장탱크 상태 확인
  • 탱크 용량·압력 수치 연동​
  • 상태에 따라 탱크 색상 변화
    (정상=청색, 주의=노랑, 경고=적색)
  • 예시: 게이지 위 실시간 수치 + 색상 동적 변화

배관 밸브 자동 제어

03

배관 밸브 자동 제어
  • 유량 시뮬레이션 애니메이션 재생​
  • 밸브 개폐 시 시각적 반응
    (밸브 아이콘 점멸, 흐름선 활성화)
  • 예시: ‘ON’ 클릭 → 흐름 경로 점등 + 밸브 회전

충전기 조작 및 응답

04

충전기 조작 및 응답
  • 사용자가 클릭 시 충전 상태 ON/OFF 전환
  • 이상 발생 시 경고 상태
    (적색 점멸, 경고창 표시)
  • 예시: 충전 애니메이션 / 충전량 증가 효과

센서 알람 감지

05

센서 알람 감지
  • 임계값 초과 시 빨간 점멸 + 팝업 알림 발생
  • 온도, 압력, 수소농도 등 각 센서별 기준치 비교
  • 예시: 센서 위 경고 아이콘 + ‘상세 보기’ 버튼

비상상황 시뮬레이션

06

비상상황 시뮬레이션
  • 누출·화재·폭발 등 비상 시나리오 선택 실행
  • 대피 경로 애니메이션 + 음성 가이드
  • 예시: ‘훈련 시작’ 클릭 → 적색 경로 점등 + 안내 음성 재생

점검자 행동 / QR 연계

07

점검자 행동 / QR 연계
  • 현장 설비 QR 스캔 시 모바일 팝업 연동
  • 체크리스트 입력 + 실시간 영상 협업 지원
  • 예시: 점검 항목 체크/설비 영상 통화 활성화

제어센터 UI 반응

08
제어센터 UI 반응
  • 알람 발생 시 대시보드에 실시간 전파
  • 히스토리 재생/조치내역 확인 가능
  • 예시: 설비 타임라인 클릭 → 과거 이벤트 리플레이

보고서 자동 생성

09

보고서 자동 생성
  • 조치 결과, 점검 항목, 이상 이벤트가 자동 정리
  • PDF/엑셀 형식으로 즉시 출력 가능
  • 예시: 조치 결과, 점검 항목, 이상 이벤트가 자동 정리
